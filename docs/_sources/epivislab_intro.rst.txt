Epivislab is a library for visualizing epidemic simulations. Epivislab is specifically designed to work with simuations generated by `Episimlab <https://github.com/UT-Covid/episimlab>`_.

View the source code on `GitHub <https://github.com/kellypierce/epivislab>`_.

Dependencies
------------

- Python >= 3.7
- python packages as listed in `requirements.txt <https://github.com/kellypierce/epivislab/blob/main/requirements.txt>`_.

Setup
-----

Clone the repository and run `python3 setup.py` to install in your local environment, then install requirements with `pip3 install -r requirements.txt`.

Tests
-----

After installing epivislab and its dependencies, tests can optionally be run with `pytest`.
